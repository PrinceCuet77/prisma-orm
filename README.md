# Prisma ORM

## Basic Query

1. Update the age of at least three users
2. Find users whose age is less than 18
3. Find and update users whose age is greater than or equal 21

## Advanced Query

1. Find all movies in the `Sci-Fi` genre released before `2009`
2. Find users who have both reviewed a movie `AND` have movies in their `favoriteMovies`
3. Find reviews with ratings between `4` and `7` stars
4. Get the 5 most recently released movies
5. Get the top 3 movies with the highest average ratings

## Aggregate Query

1. Retrieve how many reviews exist and give the total reviews for each movie
2. Retrieve each movie's average rating
3. Find how many reviews each movie has (confusion)
4. Find who gave the minimum rating and which movie has the maximum rating
5. Find which movies have an average rating greater than 7
